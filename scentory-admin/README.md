# Scentory Admin

香水作り体験ワークショップ向けの Spring Boot 管理システムの MVP 土台です。

## 実装済み

- ログイン画面
- ダッシュボード
- 予約管理
- 顧客管理
- 顧客詳細画面
- 枠・スケジュール管理
- 商品・メニュー管理
- コンテンツ管理
- スタッフ管理
- レポート・集計
- システム設定
- アカウント
- 予約ステータスの日本語表示
- 予約キャンセル導線
- 権限別メニュー表示とルート制御

## ログイン情報

- 管理者: `admin / admin123`
- スタッフ: `staff / staff123`
- 閲覧用: `viewer / viewer123`

## 確認方法

1. `c:\academia\src\portfolio\scentory-admin` を開く
2. Maven が入っている環境で `mvn spring-boot:run` を実行する
3. `http://localhost:8080/login` にアクセスする

## 補足

- この環境には `Maven / Gradle` が入っていないため、こちらでは起動確認まではできていません。
- DB は H2 のインメモリ設定です。
- 通知、CSV、外部連携、返金処理などは次フェーズ想定です。
